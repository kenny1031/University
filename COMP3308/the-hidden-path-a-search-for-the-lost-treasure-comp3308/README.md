# The Hidden Path: A Search for the Lost Treasure (COMP3308)
The treasure map is represented as a 2-dimensional grid of square tiles. The task is to find a path from the starting point to the treasure using the following search strategies: BFS, DFS, UCS, Greedy, A* and Hill-climbing. 

Run:
```bash
python hidden_path.py --strategy B --filename example1.txt
```
